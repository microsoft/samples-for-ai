# 如何运行

1. 打开解决方案。 (默认会用Visual Studio 2017打开)

2. 设置**LR项目** 为"启动项目" (通过右击项目名称)。

3. 设置脚本**logistic_regression.py**为"启动文件" (通过右击文件名称)。

4. 右击脚本 -> 在不调试的情况下启动
