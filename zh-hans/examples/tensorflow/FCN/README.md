# Tensorflow中实现FCN

[简体中文](/zh-hans/examples/tensorflow/FCN/README.md)

The implementation of Fully Convolutional Networks in the tensorflow framework.

- FCN基于VGG19

## 文件夹

- model 文件夹是vgg19的模型。
- logs 文件夹是日志。
- datasets 目录是训练和验证数据集。

### 参考
