# Tensorflow中实现FCN

[English](/examples/tensorflow/FCN/README.md)

这是tensorflow框架中实现的完全卷积网络。

- FCN基于VGG19

## 文件夹

- model 文件夹是vgg19的模型。
- logs 文件夹是日志。
- datasets 目录是训练和验证数据集。
