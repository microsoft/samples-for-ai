# 如何运行

1. 打开解决方案。(默认会用Visual Studio 2017打开)

2. 右击想要运行的项目的名字，将它设置为"启动项目"。

3. 右击想要运行的脚本的名字，将它设置为"启动文件"。

4. 右击脚本 -> 在不调试的情况下启动
